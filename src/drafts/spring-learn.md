## 什么是 Spring ?

* 是一个非常活跃的开源框架
* 由 Rod Johnson 开发
* 帮助分离项目组件之间的依赖关系
* 它的主要目的是简化企业开发

## 核心概念

* IoC (控制反转)
  * Inversion of Control
  * 对象创建和对象之间关系维护的控制权由程序员管理变为由 Spring 管理

* DI (依赖注入)
  * Dependency Injection
  * 对象和对象之间依赖关系的创建叫依赖注入

* AOP (面向切面编程)
  * Aspect Oriented Programming

## Spring 组成

* Core Container
  * Beans
  * Core
  * Context (容器)
  * SpEL (表达式语言)

* Data Access / Integration
  * JDBC
  * ORM
  * OXM
  * JMS
